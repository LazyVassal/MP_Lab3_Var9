# Файловый менеджер на Python

Простой файловый менеджер с графическим интерфейсом, реализованный на Python с использованием Tkinter.

## Возможности
- Просмотр содержимого папок.
- Создание новых папок.
- Открытие папок через диалог.
- Сохранение и загрузка состояния (последняя открытая папка) в JSON-файл.

## Требования
- Python 3.x
- Библиотека `tkinter` (входит в стандартную поставку Python)

## Запуск
1. Установите Python (если не установлен).
2. Клонируйте репозиторий или скачайте файлы.
3. Запустите приложение:
   ```
   python main.py
   ```
